Synopsis

The objective for this assignment is to predict the manner in which the subjects did the exercise. The quality of the exercise is represented by the “classe” variable in the training set. The data provided for this project consists of the sensor readings from 6 participants who performed 10 repetitions of Unilateral Dumbbell Biceps Curl in five different fashions: exactly according to the specification (Class A), throwing the elbows to the front (Class B), lifting the dumbbell only halfway (Class C), lowering the dumbbell only halfway (Class D) and throwing the hips to the front (Class E). Class A corresponds to the specified or correct execution of the exercise, while the other 4 classes correspond to common mistakes.

During this project, we explore, clean-up and understand the data to determine what specific predictor variables are likely to influence on the prediction on “classe” variable which represent exercise quality and takes on values of “A, B, C, D, or E.” We remove all the columns from the data set which are non-sensory data and only identify the different participants and timestamps of the data collections. Additionally, we also remove all the data columns which are empty or have NA values. We then conduct exploration on the remaining data to identify any predictor variables which does not show any relationship with the response variable “classe”. We specifically could not identify any such variable and thus leave all the variables, 52 of them to fit the model.

Data Cleanup and Preprocessing

We determine that the first seven columns of the data sets (both training and testing) have only identifying information which will not be needed in fitting a predictive model). We remove these columsn from the data sets. Next we identify that several data columns either have NAs or missing values and won’t contribute anything to the model. We selectively remove them from the data sets. Thus we are left with 52 predictor variables as well as the response variable “classe”, a total of 53 columns.

Create training, test and validation data sets

We take the provided training data set (which has been cleaned and preprocessed as described earlier) and divide the data into three sets - 60% into training set, 20% into testing set, and 20% into the validation set.

Training set will be used for training the model and then the model will be fine tuned against testing data set. Finally, after the model is ready, its accuracy will be tested against validation set.
